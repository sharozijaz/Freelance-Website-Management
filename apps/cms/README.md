# CMS App Legacy Notice

`apps/cms` is a V1 Payload CMS application.

For Sharoz Platform V2, this app is **LIMITED USE / TRANSITIONAL**. It may remain available for V1 content, migration support, and historical compatibility, but new V2 features must not depend on Payload page layout blocks, Payload navigation presentation, Payload site settings as public website presentation, or Payload as the Platform API.
